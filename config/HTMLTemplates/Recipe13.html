Overview

WebDisplay give you the possibility to define a page hierarchy. With this feature you can easily define a site map with different sections. For example a section for all users and another for private users only, where this different sections have different page layouts. The page layout is inherited to the subpages.

Preconditions

  • You have a well configured base WebDisplay system.

Steps to do:

  1. Define the different sections with the different pages.
  2. Write the data to Config.any: 
    [[#wd DisplayAnythingRenderer {
	/AnythingInfo {
	"/Pages {"
	"	# subclasses for "Page""
	"	/Page {"
	"		/OpenSection { "HomePage" "Welcome" }"
	"		/PrivateSection { "HomePage" "Booking" "Holidays" }"
	"}"
	}
}]]
