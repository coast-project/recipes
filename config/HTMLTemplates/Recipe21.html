Overview

We would like to render our logo in an image tag and show it on each of our Web pages.

Preconditions

  • We know what the image tag supports and what kind of options there are
  • We have got our logo in a gif-format
  • We have already some well configured pages

Steps to do:

  1. Define a image renderer specification.
    Sample: 
    [[#wd DisplayAnythingRenderer {
	/AnythingInfo {
		"/ShowImage1 {"
		"	/Image {"
		"		/ImageName "somelogo_small.gif""
		"		/Options {"
		"			"Border=2""
		"		}"
		"	}"
		"}"
	}
}]]
  2. Try out formatting and layout options.
    Sample: 
    [[#wd DisplayAnythingRenderer {
	/AnythingInfo {
		"/ShowImage2 {"
		"	/Image {"
		"		/ImageName "somelogo_small.gif""
		"		/Options {"
		"			"Border=0" "height=105" "width=365" "hspace=15""
		"		}"
		"	}"
		"}"
	}
}]]
  3. If you liked to have the same image on all of your WebDisplay powered pages, best would be to put the renderer specification in your shared-page-layout.
    Sample:
    See our itopia logo in Page.any: 
    [[#wd DisplayAnythingRenderer {
	/AnythingInfo {
		"..."
		"[[#wd Image somelogo_small.gif ]]"
		"..."
	}
}]]
